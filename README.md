# How to install on your machine

Need nodeJs 12.0+, clone the repo and create `.env` file in root dir and use the `env.example`.
After that install all needed packages write `sudo npm install`, for windows `npm install`, you can learn more about what packages i used on ðŸ‘‰ `package.json`.
run the code using `npm run dev` or `npm run start`
